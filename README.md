# Wallet-Activity-Tracker
Compact web application built with standard HTML, CSS, and vanilla JavaScript. It demonstrates how an interactive browser project can be created without external libraries or frameworks. The application allows users to organize simple records and explore dynamic interface updates.

The interface is inspired by everyday note management and naturally references Bitcoin and Wallet concepts as sample categories for entries. The project focuses on clean structure, responsive design, and readable code. It provides a practical starting point for experimenting with DOM manipulation, event handling, and client-side data management while keeping everything inside a single HTML file.
